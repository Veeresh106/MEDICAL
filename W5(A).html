Color Reference in CSS

Named Color: Red background with white text.
Hexadecimal Color: Green background with blue text.
RGB Color: Red background with yellow text.
RGBA Color: Semi-transparent blue background with semi-transparent white text.
HSL Color: Green background with red text.
HSLA Color: Semi-transparent blue background with semi-transparent yellow text.
CurrentColor Keyword: Purple text with a border using the currentColor.
